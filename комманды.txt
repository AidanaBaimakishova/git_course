
Вот список основных команд Git для работы с репозиторием:

git init: Инициализация нового репозитория Git в текущем каталоге.

git clone [URL]: Клонирование существующего репозитория Git с указанным URL.

git status: Проверка состояния файлов в вашем репозитории.

git add [file]: Добавление изменений в индекс для последующего коммита. Можно также использовать git add ., чтобы добавить все измененные файлы.

git commit -m "[commit message]": Создание коммита с добавленными изменениями и комментарием к ним.

git push [remote] [branch]: Загрузка локальных изменений в удаленный репозиторий.

git pull [remote] [branch]: Получение изменений из удаленного репозитория и объединение их с локальными.

git branch: Просмотр списка локальных веток.

git branch [branch_name]: Создание новой ветки.

git checkout [branch_name]: Переключение на другую ветку.

git merge [branch]: Объединение указанной ветки с текущей.

git remote -v: Просмотр списка удаленных репозиториев и их URL.

git log: Просмотр истории коммитов.

git diff: Просмотр различий между текущим состоянием файлов и последним коммитом.

git reset --hard [commit_id]: Возврат к определенному коммиту и удаление всех последующих изменений.

git rm [file]: Удаление файла из репозитория и индекса.

git mv [old_path] [new_path]: Переименование файла или перемещение его в другую директорию.

git stash: Скрытие локальных изменений, чтобы временно переключиться на другую задачу.

git stash pop: Применение ранее скрытых изменений.

git config: Настройка параметров Git, таких как имя пользователя, адрес электронной почты и т. д.

Это основные команды Git, которые помогут вам начать работу с системой контроля версий.
